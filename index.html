I have created this code with great difficulty over the course of seven days. This code was developed based on a friend's request, and the JavaScript code has been optimized. It's functioning correctly, so I thought I would share it with friends. In this code, I have added many options, which you can read about in detail below.

1. Main Home Screen

The current home screen contains an analog clock, a digital clock, the current Gregorian date, the current Hijri date, and the current Punjabi date, all placed on a single page for easy reference to the current time and date.

In addition, the next page also displays more timers, indicating the time that has passed since midnight and the time remaining until midnight. Furthermore, there's a leap year indicator to easily determine if the current year is a leap year or not. Additionally, both Hijri and Gregorian dates are provided in the Urdu language for your convenience.

2. Age Calculator

Furthermore, an age calculator has been added, allowing you to easily view the total elapsed time of your life. In this age calculator, you can select a date, and it will show you which day of the week it was on that date. You can also see when your next birthday will be and on which day of the week. Additionally, you can view how many years, months, weeks, days, hours, minutes, seconds, and milliseconds have passed in your life, as well as how much time is left until your next birthday.

Additionally, on the next page, you can also see the total number of days in each month for the current year, along with the name of the current month and the current date.

3. Gregorian to Hijri Calculator

With the Gregorian calendar and Hijri calendar calculator, you can easily convert Gregorian dates to Hijri dates and vice versa. It provides you with the date in both calendars, including the month, day, year, and the day of the week.

On the next page, you can see how many days have passed and how many are remaining in the current year. You can also find out the current day's number within the year. Additionally, you can determine which day of the week is the first and last day of the current year, as well as the first and last day of the current month. Furthermore, you can check how many weeks have passed and how many are left in the current year, along with the week number for the current week.

4. Text Web Tools

1. Blank Line Remover Button: Removes blank lines from the text.

2. Leading Space Remover Button: Removes extra leading spaces from lines of text.

3. Normalize Button: Normalizes words by reducing extra spaces and optimizing their placement.

4. Indent Code Button: Indents HTML code and CSS code.

5. List Maker Buttons: Adds and removes bullet or numbered lists.

6. Comment Removal Buttons: Removes comments from HTML, CSS, and JavaScript code.

7. Beautify Button: Beautifies HTML code to make it more readable and organized.

8. Copy Button: Copies selected code to the clipboard.

5. Tic Tac Toe Game

1. "Number of Games Won" Option: This option tells you how many games each team has won.

2. "Number of Games Drawn" Option: This option informs you about the number of games that have ended in a draw.

3. "Whose Turn to Play" Option: This option indicates whose turn it is to play during the game.

4. "Play Against the Computer" Option: You can choose this option to play against the computer.

5. "Vibrate on Win and Draw" Option: This option enables vibrating the screen as a notification when you win or the game ends in a draw.

6. "Highlight All Moves of the Winner" Option: This option highlights all moves (or turns) of the winning player.

7. "Restart Game Button": This button restarts the game, but the count numbers remain the same.

8. "New Game Button": This button starts a new game with all count numbers reset to zero.

9. "Undo Button": This button is added to eliminate incorrect moves.

    6. Animation Cleaner & Text Decoration

    You can use an animation cleaner to paste a file from your animation into a text area and click a button. This action will remove all code and lines such as @webkit-keyframes, -webkit, -mz, etc., making the file simpler and easier to work with."
